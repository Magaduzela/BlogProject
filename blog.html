Full Stack Web and Software Developer


A full-stack web developer is a programmer that builds a variety of web-based software solutions to solve business problems. The Full-Stack Developer’s scope of work includes the development of apps, websites, databases and desktop solutions. AIE students following this track will learn to use a variety of tools and languages to develop both front-end and back-end (server) software components.

Microsoft Developer


A Microsoft developer is a computer programmer or software engineer who has earned certification through Microsoft. The Microsoft environment provides a rich platform for developing a variety of applications using Microsoft tools, and allows programmers to deploy their solutions to the cloud using Microsoft Azure cloud platform. You will focus on developing your Object-Oriented Programming skills using Microsoft C#, and learn how to develop various types of solutions including desktop, web and cloud-based solutions.
